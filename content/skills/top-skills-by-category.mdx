## FAQ

**Q：下載量高嘅 Skill 係咪一定更安全？**

唔係。ClawHavoc 事件入面有些惡意 Skill 嘅下載量就係靠刷高嘅。下載量可以作為參考，但唔係安全保證。必須同時睇 Security Scan + 作者信譽 + Permission 要求。

**Q：`capability-evolver` 嘅 35K 下載量係咪可信？**

呢個 Skill 確實有很高嘅用戶基礎，但「AI 自動進化自身」係非常高風險嘅能力。建議先用幾週後再決定要唔要裝，唔要因為下載量高就覺得安全。

**Q：官方 Skills 有冇可能出問題？**

極少，但唔係零。官方 Skill 有 OpenClaw 團隊審核，風險遠低於社區 Skill。如果你係高度安全意識嘅用戶，`skills.verified_only true` 係最保守嘅設定。
