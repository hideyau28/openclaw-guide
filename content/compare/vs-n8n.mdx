## 三款工具嘅本質分別

### OpenClaw：AI 做判斷

你用自然語言話你想要乜結果，AI 自己決定點做。

```
你：「監控 GitHub repo，有新 PR 就幫我 review 同分類優先級」
OpenClaw：自主讀 PR → 理解代碼 → 撰寫 review → 按重要性分類 → 通知你
```

**強項：** 處理模糊指令、理解上下文、做出判斷。
**弱項：** 唔確定性——AI 有時做錯決定。

### n8n：你設計流程

你用視覺化界面設計流程圖，系統嚴格按流程執行。

```
Trigger（新 PR）→ 讀 PR metadata → 如果 label = "urgent" → 發 Slack 通知
                                  → 否則 → 加到 queue
```

**強項：** 確定性，每次都一樣。可自建，完全免費。
**弱項：** 冇預設過嘅情況就做唔到。

### Zapier：最簡單嘅串接

類似 n8n，但更注重易用性。大量預建模板，唔識技術都可以上手。

**強項：** 最易用，5000+ 整合。
**弱項：** 價格最高，數據存喺 Zapier server。

## 真實場景對比

想像你收到一封客戶 email：「我上星期買嘅嗰個嘢好似有啲問題，可以幫我睇睇嗎？」

### Zapier 做法：
```
收到 email → 檢查有冇 keyword「問題」→ 轉發去 support@
```
問題：keyword matching 太死板，好多變化佢識別唔到。

### n8n 做法：
```
收到 email → 正則表達式匹配 → 分類 → 發通知
```
好啲，但仍然需要你預設所有匹配規則。

### OpenClaw 做法：
```
收到 email → AI 理解：客戶有產品問題
→ 查 CRM 睇訂單歷史
→ 判斷優先級：非緊急
→ 自動回覆：確認收到 + 預計處理時間
→ 建立 support ticket
→ Telegram 通知你
```

OpenClaw 嘅優勢係佢「理解」呢封 email 嘅意圖，而唔係靠 keyword matching。

## 最佳組合：OpenClaw + n8n

Gemini Deep Research 嘅分析確認：**AI Agent + 確定性 Workflow 係目前業界最佳實踐。**

原理好簡單：
- **OpenClaw** 做需要 AI 判斷嘅部分（分析、理解、決策）
- **n8n** 做確定性嘅執行部分（數據同步、格式轉換、通知轉發）

```
實戰範例：

n8n：監控新 GitHub PR（Webhook trigger）
  ↓
n8n：呼叫 OpenClaw Gateway API
  ↓
OpenClaw：AI 分析 PR 代碼 + 撰寫 review + 分類優先級
  ↓
n8n：根據 OpenClaw 嘅結果
  → 緊急 → 即刻發 Slack 通知
  → 普通 → 加到 daily digest
  → 有安全問題 → 發 email 畀 security team
```

呢個組合利用每款工具嘅強項，避免各自嘅弱點。AI 做判斷，workflow 做執行——可靠又靈活。

自託管 n8n 同 OpenClaw 嘅最佳方式係放上 VPS。[DigitalOcean](https://m.do.co/c/7692f5f6464a) 嘅 $12/月 Droplet 或者 [Vultr](https://www.vultr.com/?ref=9882956-9J) 嘅 $6/月 VPS 已經夠跑兩者。

## OpenClaw 嘅優勢同限制

**優勢：**
- 自然語言設定，唔需要學 workflow 概念
- AI 判斷能力，處理模糊指令
- 靈活應對新情況
- 本地運行，數據自己控制

**限制：**
- AI 可能做錯決定（唔確定性）
- 每次呼叫都有 API 費用
- 安全風險——本地 Agent 有完整系統權限，詳細睇[安全避坑指南](/security/essential-guide)
- 較難 debug（AI 做錯咗有時唔知點解）

## n8n 嘅優勢同限制

**優勢：**
- 確定性執行，每次都按設計運行
- 自建完全免費
- 400+ 整合，主流服務都有
- 社區大，大量現成 workflow 模板

**限制：**
- 每個情況需要預先設計

## FAQ

**Q：可唔可以 OpenClaw、n8n、Zapier 同時用？**

完全可以。好多人嘅組合係：OpenClaw 做 AI 判斷 → n8n 做執行流程 → Zapier 做少量簡單串接（因為 Zapier 某啲 App 嘅整合比 n8n 做得好）。唔需要只揀一個。

**Q：n8n 免費版夠唔夠用？**

自建版係完全免費，功能同雲端版一樣，只係你要自己維護 server。如果你識基本嘅 Docker 操作，自建 n8n 唔難——10 分鐘搞掂。如果唔想自己管 server，n8n Cloud 最平 €20/月。

**Q：我完全唔識技術，應該揀邊個？**

Zapier。門檻最低，大量模板，拖拉就搞掂。如果你嘅需求係「Gmail 收到訂單 email 自動記入 Google Sheets」呢類確定性任務，Zapier 最適合。OpenClaw 係畀想要 AI Agent 嘅人，唔係想要 workflow builder 嘅人。

**Q：OpenClaw 同 n8n 點樣整合？**

n8n 有 HTTP Request node，可以呼叫 OpenClaw Gateway API。反過來，OpenClaw 可以呼叫 n8n 嘅 webhook trigger n8n workflow。設定指引可以參考 [WhatsApp 自動化教學](/use-cases/whatsapp-assistant) 入面嘅多工具整合章節。
