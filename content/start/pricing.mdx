## TL;DR

- **OpenClaw 軟件：免費開源**
- **AI Model API：$5–50/月（視乎用量同模型）**
- **VPS 部署（可選）：$5–20/月**
- **最大隱藏陷阱：長 session 累積 token，不知不覺燒錢**

## 費用組成：4 個部分

### 1. OpenClaw 軟件 = 免費

OpenClaw 係開源軟件，下載、安裝、使用全部免費。冇任何訂閱費、授權費。

### 2. AI Model API = 按 token 計費（主要開支）

每次 Agent 呼叫 AI 都有 API 費用。你用幾多付幾多，冇最低消費。費用取決於你選嘅模型同用量。

### 3. VPS 雲端部署 = $5–20/月（可選）

如果你想 OpenClaw 24 小時運行（例如做 WhatsApp 自動回覆），需要一部 VPS。喺自己部 Mac 跑就唔使呢個費用。

### 4. 搜索 API = 免費 tier 夠用

Brave Search、Tavily 等搜索 API 都有免費額度，正常用量完全唔使付錢。

## 最大隱藏陷阱：Session History 累積

<Callout variant="warning">
**呢個係最多人踩嘅坑。**

同一個 session 嘅所有對話歷史，每次呼叫 API 都會全部重新傳送。Session 越長，每次呼叫嘅 token 數越多，費用指數級增長。

MIT Technology Review 報導過有用戶用 Claude Opus 4.5 兩個鐘燒咗 **$100 美金**——唔係 Bug，純粹係 session 太長。

**解決方法：** 每個獨立任務打 `/new` 開新 session。詳細慳錢策略睇 [Token 慳錢攻略](/troubleshoot/token-optimization)。
</Callout>

## 慳錢 5 大策略

1. **每個任務開新 session** — 打 `/new`，避免 history 累積
2. **按任務揀模型** — 簡單任務用 Haiku/DeepSeek，複雜任務先用 Sonnet
3. **設 API 用量上限** — 喺 provider dashboard 設 hard limit，防止意外
4. **善用免費額度** — OpenAI Codex OAuth 免費、Gemini 免費 tier、DeepSeek 新帳號送額度
5. **監控 token 消耗** — TUI 底部嘅 token 計數幫你即時追蹤

詳細方法睇 [Token 慳錢攻略](/troubleshoot/token-optimization)。

## FAQ

**Q：有冇完全免費嘅方法用 OpenClaw？**

有。OpenAI Codex OAuth 有免費額度（唔需要信用卡），Google Gemini 有免費 tier，DeepSeek 新帳號有少量免費額度。用呢幾個組合試玩 OpenClaw 完全唔使畀錢，只係用量有限制。詳細申請方法睇 [API Key 設定指南](/install/api-keys)。

**Q：最平可以幾錢？**

如果你有 Apple Silicon Mac，用 Ollama 跑本地模型完全免費（只有電費）。如果用雲端 API，DeepSeek V3 + Gemini Flash 組合，每月 $3–5 USD 可以做到輕度自動化。

**Q：點樣設預算上限？**

每個 API provider 都有用量限制功能。OpenAI 可以喺 Billing → Usage limits 設 hard cap。Anthropic 可以喺 Plans & Billing 設月度上限。OpenRouter 可以喺 Credits 設定用完即停。建議新手先設 $10/月嘅上限試玩。

**Q：費用係按月固定嗎？**

唔係。API 費用按實際使用量計算，冇最低消費。月底費用視乎你嘅 session 長度、呼叫頻率、選用模型而定。建議喺 provider dashboard 設月度上限，咁就唔會有意外賬單。
