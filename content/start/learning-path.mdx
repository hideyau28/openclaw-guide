## FAQ

**Q：路線 B 嘅 45 分鐘係真實嘅嗎？**

係平均時間。第一次設定 OpenClaw 嘅人，包括遇到小問題排查，通常需要 30–60 分鐘。如果你熟悉 Terminal 同 npm，可能 20 分鐘就完成。如果係第一次用 Terminal，可能需要 90 分鐘。不過一旦裝好，之後嘅設定都係幾分鐘嘅事。

**Q：我係 Windows 用戶，路線 B 都適用嗎？**

適用，步驟大致相同。但本站嘅[安裝教學](/install/macos)係 macOS 版本，Windows 版本有少量指令差異（例如用 `choco` 代替 `brew`）。建議對照官方 OpenClaw 文檔嘅 Windows 部分。

**Q：路線全部都係連去呢個網站嘅文章嗎？**

係嘅，全部係本站文章，確保內容同版本同步。如果你需要官方技術文檔（例如 API reference、完整 config 文件），可以去 OpenClaw 官方 GitHub repo。
