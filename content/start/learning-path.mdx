## TL;DR

4 個階段，由完全唔識到識得自動化：

1. **認識**（30 分鐘）— 了解係乜、幾錢、同 ChatGPT 有乜分別
2. **安裝 + 安全**（2 小時）— 裝軟件、設 API、做安全設定
3. **進階設定**（1 小時）— 自定義 Agent、裝 Skills、控制費用
4. **排錯 + 實戰** — 遇到問題時參考

唔需要全部睇完先開始。跟住你嘅階段走就得。

---

## Stage 1 — 認識（30 分鐘）

<Callout variant="tip">
**目標：了解 OpenClaw 係咪適合你，值唔值得花時間安裝。**
</Callout>

按呢個順序讀：

### 1. 了解 OpenClaw 係乜

[OpenClaw 係乜？5 分鐘懶人包](/start/what-is-openclaw) — 用最簡單嘅語言解釋 OpenClaw 係乜、可以做啲乜、有乜風險。**5 分鐘讀完就有基本概念。**

### 2. 知道要花幾多錢

[費用完整拆解](/start/pricing) — OpenClaw 軟件免費，但 AI Model API 要錢。呢篇幫你搞清楚每月大約使幾多。**唔好到裝完先嚟驚費用。**

### 3. 理解同 ChatGPT 嘅分別

[OpenClaw vs ChatGPT 完整對比](/compare/vs-chatgpt) — 如果你而家用緊 ChatGPT，呢篇幫你理解 OpenClaw 嘅定位同 ChatGPT 完全唔同。一個係 chatbot，一個係 agent。

## Stage 2 — 安裝 + 安全（2 小時）

<Callout variant="info">
**目標：裝好一個安全嘅 OpenClaw 環境，可以開始用。**
</Callout>

### 1. 安裝 OpenClaw

[macOS 完整安裝教學](/install/macos) — 一步步跟住截圖做，包括 Homebrew、Node.js、OpenClaw 安裝同 Gateway 啟動。**約 20–30 分鐘。**

### 2. 設定 API Key

[API Key 申請與設定指南](/install/api-keys) — 搞掂你嘅 LLM provider（OpenAI、Anthropic、DeepSeek、OpenRouter、Google）。**一個 Key 就可以開始用。**

### 3. 做安全設定

[安全避坑指南](/security/essential-guide) — **呢步最重要，唔好跳過。** OpenClaw 有你部機嘅完整操作權限，安全設定做唔好等同將控制權交出去。

<Callout variant="warning">
**唔好跳過安全設定。** ClawJacked 漏洞（CVSS 8.8）已經影響 13.5 萬實例，ClawHub 上 8% 嘅 Skills 係惡意嘅。呢啲唔係理論風險，係真實發生緊嘅事。
</Callout>

## Stage 3 — 進階設定（1 小時）

<Callout variant="tip">
**目標：將 OpenClaw 調校成你嘅專屬助手，同時控制好費用。**
</Callout>

### 1. 自定義 Agent 人格

[soul.md 完整教學](/start/soul-md) — soul.md 係 OpenClaw 嘅「性格檔案」。設定好之後，Agent 會用你想要嘅語氣、風格、規則嚟做嘢。

### 2. 安裝推薦 Skills

[Skills 安全安裝指南](/skills/essential-setup) — Skills 係 OpenClaw 嘅擴展功能。呢篇教你點樣安全地揀同裝 Skills，避開惡意 Skills。

### 3. 控制費用

[Token 慳錢攻略](/troubleshoot/token-optimization) — 學識點樣監控 token 消耗、按任務揀模型、設預算上限。**唔做呢步可能會燒錢。**

## FAQ

**Q：全部跑完要幾耐？**

Stage 1 + 2 大約 2.5 小時，之後你就有一個安全、可以用嘅 OpenClaw 環境。Stage 3 可以之後慢慢做。如果你熟悉 Terminal 同 npm，可能 1.5 小時就搞掂。

**Q：我係 Windows 用戶，呢條路線都適用嗎？**

適用，步驟大致相同。但本站嘅[安裝教學](/install/macos)係 macOS 版本，Windows 版本有少量指令差異（例如用 `choco` 代替 `brew`）。安全設定同 API Key 設定嘅步驟完全一樣。

**Q：可唔可以跳過 Stage 1 直接裝？**

可以，但我建議至少快速掃一下 [費用拆解](/start/pricing)。好多人裝完先嚟驚 API 費用，其實提前知道就唔會有意外。

**Q：路線入面嘅文章會唔會過時？**

所有文章都標明咗 lastVerified 日期同 openclawVersion。我哋會定期更新確保同最新版本同步。如果你發現任何過時內容，歡迎透過 [About 頁面](/about) 聯絡我哋。
