## OpenClaw 係乜？一句話解釋

> **ChatGPT 你問佢答，OpenClaw 佢自己識做嘢。**

ChatGPT 係 chatbot，你問佢問題，佢回答你。OpenClaw 係 AI Agent——你畀佢一個任務，佢自己去完成。佢可以讀你嘅檔案、執行指令、發 WhatsApp 訊息、打開瀏覽器做嘢，全部自動進行。

最關鍵嘅分別：**OpenClaw 跑喺你自己部電腦**，數據唔離開你嘅機。詳細比較睇 [OpenClaw vs ChatGPT 完整對比](/compare/vs-chatgpt)。

## OpenClaw 可以做啲乜？5 個真實例子

### 1. WhatsApp 24 小時自動回覆客人

你瞓緊覺，客人喺 WhatsApp 問你產品價錢。OpenClaw Agent 自動回覆準確嘅報價，第二朝你起身先知有人問過嘢——已經處理好咗。

### 2. 每朝自動整理 Email 摘要推送去 Telegram

朝早返工之前，OpenClaw 已經將你收件箱入面嘅重要 email 整理成摘要，喺 Telegram 發畀你。你未打開電腦已經知道今日要處理啲乜。

### 3. 監控網站價格變動自動通知

你想買嗰個 AirPods，OpenClaw 每隔 15 分鐘幫你 check 價錢。跌到你嘅目標價就即刻 Telegram 通知你。

### 4. 自動 Review GitHub PR

團隊有人開咗新 PR，OpenClaw 自動跑一次 code review，喺 PR 留下具體嘅改善建議。你做 reviewer 嘅時間慳咗一半。

### 5. 控制你部電腦嘅瀏覽器做重複性工作

每個禮拜你都要登入 5 個唔同嘅 dashboard check 數據？OpenClaw 用瀏覽器自動化幫你做，將結果整理成報告。

## 安全風險：裝之前必須知

<Callout variant="danger">
**OpenClaw 唔係零風險。** 一個本地 AI Agent 有你部機嘅真實操作權限，以下係已知嘅重大風險：

- **CVE-2026-25253（ClawJacked，CVSS 8.8）：** 舊版 Gateway 可被惡意網站遠端接管。全球 13.5 萬實例暴露喺公網。**立刻更新至 v2026.2.26+。**
- **ClawHub 惡意 Skills：** 10,700 個 Skills 入面有 820+ 被 Koi Security 標記為惡意（約 8%），可以植入 AMOS Stealer 偷你嘅密碼。
- **Gateway Token 外洩：** Token 等同你 Agent 系統嘅鎖匙，外洩即代表完全失控。

**裝之前一定要睇：[安全避坑指南](/security/essential-guide)**
</Callout>

## FAQ

**Q：OpenClaw 跑喺邊度？需唔需要 internet？**

OpenClaw 軟件本身跑喺你自己部機（本地）。但每次 Agent 需要 AI 回應時，都要連去你設定嘅 LLM provider（例如 OpenAI 或 Anthropic）。如果你用本地模型（Ollama），就可以完全離線運行。

**Q：OpenClaw 支唔支持 Windows？**

支持 Mac、Windows 同 Linux。但目前社區最多教學係 macOS 版本，Windows 版本偶爾有額外步驟。本站嘅[安裝教學](/install/macos)係 macOS 版本。

**Q：OpenClaw 同 Cursor / GitHub Copilot 有乜分別？**

Cursor 同 Copilot 係代碼編輯器嘅 AI 助手，專注喺寫代碼。OpenClaw 係通用 AI Agent——佢唔止寫代碼，仲可以發訊息、管理檔案、控制瀏覽器、做任何可以用電腦做嘅事。想了解更多工具比較，睇我哋嘅[推薦工具頁面](/resources)。

**Q：我完全唔識技術，可以用 OpenClaw 嗎？**

老實講，OpenClaw 需要基本嘅 Terminal 操作能力。如果你完全唔識但又想用，可以考慮我哋嘅[代客安裝服務](/services/secure-setup)，30–60 分鐘幫你遠程裝好。
